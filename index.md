# ToIP Main Glossary

This site publishes the **Governance-Executable Glossary** for Trust Over IP concepts.

It combines human-readable glossary pages with machine-readable governance metadata so the terminology can support implementation, audit, assurance, and interoperability work.

## What you can do here

- browse glossary terms rendered from the structured YAML source layer
- review governance-oriented documentation and improvement guidance
- download machine-readable bundles for tooling and analysis
- use a GitHub-native documentation workflow built on Jekyll and Just the Docs

- [Glossary Terms](/terms-index/)

## Governance-executable model

Each term can include operational metadata such as authority scope, delegation mode, revocation support, lifecycle states, evidence expectations, and control-plane decision points.

That makes the glossary useful not only as a reading resource but also as a structured governance substrate.
